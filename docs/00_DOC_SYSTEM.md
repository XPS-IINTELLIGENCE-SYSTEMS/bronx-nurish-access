# Bronx Nourish Access — Production Documentation System

Status: Operational MVP verified on 2026-05-09.

## Documentation standard

This repository uses a production handoff structure modeled after common GitHub project standards:

- README: public project overview and live URLs.
- Architecture: system map, app flow, database, APIs.
- Operations: daily launch and maintenance procedures.
- Analytics: UTM, reporting, Neon tables, proof logs.
- Launch Strategy: campaign economics, 80-day reach model, channel plan.
- Brand Assets: logo, PWA icon, banners, digital/print sizes.
- Compliance: safe claims, blocked claims, consent, privacy rules.
- Investor Package: client-facing narrative and offer structure.

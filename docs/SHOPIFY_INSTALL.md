## Shopify steps

1. Open Shopify Admin.
2. Go to Online Store > Themes.
3. Duplicate the active theme.
4. Open Edit code on the duplicate theme.
5. Add a new section named `bna-image-funnel-app.liquid`.
6. Paste the section file from this repo.
7. Add a new template named `page.food-help-image.json`.
8. Paste the template file from this repo.
9. Create a page titled `Bronx Nourish Access - Food Help Eligibility Funnel`.
10. Assign the `food-help-image` page template.
11. Set the handle to `/pages/food-help`.
12. Keep the page unpublished until approved.

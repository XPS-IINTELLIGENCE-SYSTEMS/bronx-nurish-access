# Bronx Nourish Access Prelaunch QA Checklist

Status: draft only until explicit publishing approval.

## Visual QA

- Desktop hero image loads.
- Mobile hero image loads.
- CTA overlay aligns with the poster image.
- Buttons are readable over the image.
- CTA cards do not cover critical poster content.
- Mobile sticky bar does not block form fields.

## CTA QA

- Main CTA opens or scrolls to form.
- Request callback CTA works.
- Call button uses verified phone number.
- Spanish CTA is reviewed before launch.
- UTM parameters are preserved.

## Form QA

Required fields:

- First name
- Phone
- ZIP code
- Medicaid or health plan status
- Need help getting groceries or cooking
- Consent to contact

Optional fields:

- Preferred language
- Best callback time

Do not collect:

- SSN
- Medicaid ID
- case number
- immigration status
- diagnosis
- private documents

## Compliance QA

Confirm before publishing:

- program name
- payer or sponsor
- meal quantity if mentioned
- Bronx ZIP restriction
- government-paid language
- Medicaid wording
- privacy policy
- consent language
- Spanish translation

## Claims QA

Do not use unless verified:

- guaranteed free meals
- guaranteed approval
- 3 meals per day
- federal government paid
- 84 days
- 90 days
- official government program

## Shopify QA

- Section file exists in duplicate theme.
- Page template exists.
- Page handle is `/pages/food-help`.
- Page is unpublished until approved.
- Form notification destination is confirmed.
- Theme preview is approved on desktop and mobile.

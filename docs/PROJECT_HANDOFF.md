# Bronx Nourish Access Funnel App Handoff

Status: draft and staging only.

## Repo

## Purpose

Food-help routing funnel for Bronx Nourish Access.

The build uses the full landing page image as the visual base and places live buttons, text, form behavior, and mobile sticky actions over the image.

- `docs/SHOPIFY_INSTALL.md` - Shopify install guide

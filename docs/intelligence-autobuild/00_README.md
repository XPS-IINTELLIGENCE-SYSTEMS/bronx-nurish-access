# Intelligence + AutoBuild Pre-Production Documentation

This folder contains the documentation-only package for the Intelligence OS and AutoBuild OS implementation branch.

Branch: `autonomous-gpt-bridge`

Scope:
- Architecture documents
- Agent specifications
- Workflow playbooks
- Test harness design
- Simulation plan
- Benchmark criteria
- Template catalogs
- Release and validation gates

Non-scope:
- No production code
- No secrets
- No live publishing
- No destructive automation

Primary goal:
Prepare the repository so the build team can implement the GPT bridge, Vercel scheduler, sandbox, agents, validation, and Shopify workflow from approved specifications.
